This is ZK Spreadsheet for XPages demo database.
To run this demo application, you have to install ZK Spreadsheet for XPages plugin into your designer and server.
Please download plugin from one of following sites : 
	OpenNTF : http://www.openntf.org/internal/home.nsf/releases.xsp?action=openDocument&name=ZK%20Spreadsheet%20for%20XPages&documentId=null
	Google code : http://code.google.com/p/zkxpage/downloads/list

Installation steps at :  
	http://code.google.com/p/zkxpage/wiki/ZK_Spreadsheet_for_XPage_Installation
